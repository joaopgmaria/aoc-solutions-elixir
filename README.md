# AdventOfCode solutions in Elixir

**An attempt at [Advent Of Code](https://adventofcode.com/) using Elixir**

## Run

```cmd
iex -S mix
```

```elixir
Day1.part1()
Day1.part2()
...
Dayn.partn()
```

## Test

Run all tests
```cmd
mix test
```

Run exercise (day) specific tests
```cmd
mix test --only day:1
```
